when
****

A wrapper library for date/time conversion.  Takes many of the standard ISO
date formats as strings and creates an internally consistent datetime object.

Installation
============

.. code-block:: bash

    $ pip install pywhen

Supports
========

pywhen has been tested with Python 2.7, 3.5, and 3.6
